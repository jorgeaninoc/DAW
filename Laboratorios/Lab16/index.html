¿Por qué es una buena práctica separar el modelo del controlador?

Porque aumenta el nivel de reutilización de código en la aplicación, facilita modificaciones, también aumenta la cohesion del código y disminuye el acoplamiento. También si algo falla puedes saber donde estuvo el error más facilmente.

¿Qué es SQL injection y cómo se puede prevenir?

Sucede cuando se le pide al usuario un input como un id o contraseña y el usuario ingresa una sentencia SQL que correra en la base de datos sin que el administrador del sistema sepa. Puede causar la destrucción de la base de datos o el acceso al sistema por parte del usuario con permisos de super usuario. Se puede prevenir usando prepared statements que revisen el input del usuario.
