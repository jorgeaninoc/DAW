Conclusiones

Las bases de datos investigadas estan muy bien desarrolladas, sin embargo, su uso depende para quien estemos desarrollando una pagina web con manipulación de datos. Es notable que las bases de datos investigadas son muy caras para las empresas, pero depende para quien estemos trabajando. En mi opinión no es muy grande la diferencia en precios de las investigadas, excepto en la no relacional. Tal como vi la no relacional (MongoDB) puede ser muy buena opción para un sistema que vaya a ser de uso local, la seleccione debido a que en un lugar donde yo trabajaba utilizaban esta base de datos y la verdad noté que se ahorraban mucho dinero en esto ya que el sistema iba a ser de uso privado. Fue muy interesante realizar esta investigación ya que puedes abrirte el panorama sobre las bases de datos.
