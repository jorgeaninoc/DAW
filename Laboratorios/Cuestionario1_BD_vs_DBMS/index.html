¿Qué es entonces una base de datos - DB?

Es una colección de archivos interrelacionados no necesariamente ligados a un programa o un individuo.
Los archivos en una base de datos tienen ciertas restricciones: son archivos en los que aparecen múltiples ocurrencias de un mismo tipo de registro.
Una base de datos también se puede definir como un conjunto de información relacionada que se encuentra agrupada o estructurada. [1]

¿En qué casos es conveniente usar bases de datos?

  • La información que se maneja presenta varios archivos interrelacionados.
  • Cuando se requiere información ordenada y estructurada por varios usuarios o aplicaciones.

¿Qué es un sistema de gestión de base de datos - DBMS?

Un sistema de gestión de base de datos (Database Management System DBMS) es formado por un conjunto de datos interrelacionados y un conjunto de programas que permiten acceder a esos datos. La colección o conjunto de datos es denominada una base de datos que contiene información sobre una empresa determinada.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

  • El objetivo primordial de una DBMS es proporcionar una manera eficiente para extraer y almacenar información de la base de datos.
  • Gestionan grandes bloques de información.
  • Definen estructuras para almacenar y gestionar información.
  • Mantienen la información segura, aunque el sistema se caiga, no permite accesos no autorizados.
  • Evita errores si hay varios usuarios tratando de acceder a la información.

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

En proyecto de videojuegos. Si escalabamos el proyecto a un nivel aún mayor, se podría utilizar una base de datos para guardas la información de los usuarios del juego, sus puntajes, entre otras cosas. Utilizariamos una base de datos porque si el juego tiene muchos usuarios ya que es algo público necesitariamos almacenar grandes cantidades de información para poder comparar y obtener los puntajes mas altos.
