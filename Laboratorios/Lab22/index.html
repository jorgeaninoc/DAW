En este laboratorio junté Google Calendar con mi aplicación web para registrar actividades en mi proyecto de equipo.

¿Qué ventajas y desventajas tiene la integración de tus aplicaciones web con servicios web desarrollados por terceros?

Ventajas

  • Separación cliente/servidor.
  • Agrega funcionalidades al sistema.
  • Ahorra tiempo de desarrollo.
  • Fiabilidad, escalabilidad, flexibilidad.
  • Mejora la experiencia de usuario.
Desventajas
  • Rigidez en el desarrollo, no se pueden agregar funcionalidades que tú desees tener.
  • Te obliga a salir del confort de tú sistema.
  • Pueden hacer que se pierda un poco la consistencia del sistema.
